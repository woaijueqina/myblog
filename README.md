# Django+bootstrap实现个人博客


### 版本：
 >python 3.7 
 >django 3.0 
 >bootstrap-3.3.7  
 >mysql  

### 实现功能:
 > 发布内容支持markdown格式，代码语法支持高亮  
 > 评论  
 > 点赞  
 > 分页  
 > 分类
